# oil-forecasting-pipeline
# 🛢️ Brent Crude Oil Price Forecasting Pipeline

A full end-to-end data science project forecasting Brent Crude Oil prices using Facebook Prophet.

## 🔗 Live Dashboard
[Click here to view the live dashboard](https://oil-forecasting-pipeline-2026.streamlit.app)

## 📋 Project Overview
- **Data**: 10 years of Brent Crude Oil prices (2015-2026) from Yahoo Finance
- **Model**: Facebook Prophet time series forecasting
- **Accuracy**: 87% (MAPE: 12.73%)
- **Stack**: Python, Prophet, Pandas, Streamlit, Matplotlib

## 🏗️ Pipeline Structure
- Data Collection & Ingestion
- Data Cleaning & Feature Engineering
- Exploratory Data Analysis
- Time Series Forecasting Model
- Interactive Dashboard

## 📊 Key Findings
- COVID 2020 caused the largest single price crash in the dataset
- Russia-Ukraine war 2022 drove prices to 10 year highs
- March 2026 showing elevated volatility consistent with geopolitical risk
- Model predicts mean reversion toward $70-75 from current $92 spike

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Facebook Prophet
- Scikit-learn
- Streamlit
